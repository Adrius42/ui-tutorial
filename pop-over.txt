Sometimes it is necessary to ask if something should really be done (quit, overwrite, delete, ...). For this task you can use a popover view.

1.) Please run the script (only the iPad supports popover views) and create the initial view 'pop-over'.
2.) Create another 'Script with UI' and call it 'po'.
3.) The second script should contain a label 'Really want to quit' and two buttons (yes and no).
4.) The action method of both buttons is called 'ask_user'
5.) The code of the 'pop-over' script:
